# NiftyIQ - Financial Intelligence Platform

🚀 Financial Intelligence Platform for Nifty 100 Companies.<br>

📊 12+ Years Financial Data Analysis<br>
🐍 Python ETL Pipelines (Pandas, SQLAlchemy)<br>
🗄️ PostgreSQL Data Warehouse<br>
⚙️ Django REST APIs<br>
💻 React.js Interactive Dashboards<br>
📈 Power BI Business Intelligence Dashboards<br>

## Tech Stack

Python • PostgreSQL • Django • DRF • React.js • Tailwind CSS • Power BI • Pandas • SQL

## Features

✔ Company Financial Analysis<br>
✔ Revenue & Profit Growth Tracking<br>
✔ Cash Flow Analysis<br>
✔ Balance Sheet Analysis<br>
✔ EPS & Debt-to-Equity Metrics<br>
✔ Financial Health Scoring<br>
✔ REST API Access<br>

Developed during my Data Analyst Internship at Bluestock Fintech.
